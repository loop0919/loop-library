## GitHub Pages の公開方法

1. GitHub リポジトリの **Settings > Pages** を開く
2. **Build and deployment** で `Deploy from a branch` を選択
3. Branch に `main`、Folder に `/docs` を選択
4. 保存後、数分で `https://<user>.github.io/<repo>/` で公開されます

将来的にページ数が増える場合は、MkDocs や Docusaurus への移行も検討できます。
